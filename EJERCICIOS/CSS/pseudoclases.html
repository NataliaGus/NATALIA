Pseudo-clases de CSS

Una pseudo clase se usa para definir un estado especial de un elemento.

Ejemplos:

  • Estilizar un elemento cuando pasamos el cursor del ratón por encima de este.
  • Estilizar los links visitados y sin visitar de forma diferente.
  • Estilizar un elemento cuando obtiene el foco.

Diferentes estados de los links

Esto es un link

Un link tiene 4 estados distintos: link (por defecto, sin visitar), visisted. hover (cuando pasamos el cursor por encima), active (en el momento en el que estamos clicando sobre el link).

a:hover DEBE ir después de a:link y a:visited para poder ser efectivo.

a:active DEBE ir después de a:hover para poder ser efectivo.
