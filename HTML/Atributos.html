Atributos en elementos de HTML

Todos los elemetnos pueden contener atributos. Los atributos proveen información adicional sobre los elementos.

Los atributos especifican en la etiqueta de apertura. Se componen de los partes, nombre y valor.

Name="value"

href

Es el atributo básico de los links (a). La (a) define un hipevínculo, el atributo href define la URL de la página donde nos va a llevar el hipevínculo.

Página de búsqueda

Src

La etiqueta o el elemento (img) se usa para insertar una imagen en una página html. El atributo src indica el camino o dirección de la imagen a mostrar.

Src con URL o vínvulo absoluto.

Src con URL o vínculo relativo.

Width y Height

Los atributos Width (ancho) y Height (altura) definen el tamaño que queremos otorgarle a nuestra página.

Alt

El atributo Alt define el texto alternativo para una imagen. Este texto te mostrará si por alguna razón, la imagen no puede cargarse.

Style

Este atributo se usa para añadir estilos visuales a cualquier elemento, como por ejemplo, color, fuente -font-, tamaño -size-, etc.

Click aquí para más detalles sobre el style.

Title

Este atributo se usa para proveer información extra sobre un atributo.

Esta información se muestra en forma de tooltip al dejar el cursor sobre ese elemento.
