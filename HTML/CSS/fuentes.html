Fuentes

Elegir una fuente apropiada tiene un fuerte impacto en cómo los lectores experimentan un sitio web.

La fuente correcta puede crear una reconocible entidad para tu marca o página web.

Usar una fuente fácil de leer es importante. Ya que añade valor a tu texto. También es importante elegir un correcto color y tamaño.

Familias de fuentes genéricas

En CSS hay 5 familias genéricas

  1. Fuentes con serif tienen un pequeño trazo en las puntas de cada letra.
  2. Fuentes Sans-Serif. No tiene los trazos de las anteriores y tienen un aspecto minimalista y moderno.
  3. Monospace. Todas las letras tienen la misma anchura. Crean un aspecto mecánico. La fuente que vemos en nuestro código es de este tipo.
  4. Cursivas. Imitan la caligrafía humana.
  5. Fantasy. Tienen un aspecto decorativo/juguetón.
